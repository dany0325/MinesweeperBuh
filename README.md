# 💣 Minesweeper — Terminal Edition

Classic Minesweeper built with pure HTML, CSS, and JavaScript.
No dependencies. No build step. Just open `index.html`.

## Difficulty
| Level  | Grid  | Mines |
|--------|-------|-------|
| Easy   | 9×9   | 10    |
| Medium | 16×16 | 40    |
| Hard   | 30×16 | 99    |

## Controls
| Action         | Input                     |
|----------------|---------------------------|
| Reveal cell    | Left-click                |
| Place flag     | Right-click               |
| Chord reveal   | Double-click / left-click revealed cell |
| New game       | Click the ◈ button        |

## Notes
- First click is always safe (3×3 safe zone guaranteed).
- The timer starts on your first click.
